# VaraAIEvaluation

Open-source evaluation framework for public-facing LLM apps.

Design goals:
- Model-agnostic, production-ready, deterministic where possible
- RAG with FAISS, guardrails, Playwright UI testing hooks
- Clear separation of concerns, extendable metrics

See [vara_ai_eval/cli/runner.py](vara_ai_eval/cli/runner.py) for a minimal example runner.
